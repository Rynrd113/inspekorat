🔧 TROUBLESHOOTING:

Jika ada error:
1. Pastikan Laravel server berjalan: php artisan serve
2. Pastikan database terkoneksi
3. Install dependencies: pip3 install -r requirements.txt
4. Berikan permission: chmod +x *.sh
5. Lihat log error di storage/logs/laravel.log
